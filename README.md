# Impacto de IA no trabalho

Este é um trabalho de faculdade da matéria de Tópicos de Big Data em Python.

Como projeto, resolvemos fazer um site com front-end em Vue.js e back-end em Python com PySpark, para fazermos uma análise dos dados do tema escolhido pela equipe, que é sobre o Impacto de IA no mercado de trabalho.

## Data Sets encontrados e usados no projeto:
[Ai Impact on Job Market](https://www.kaggle.com/datasets/sahilislam007/ai-impact-on-job-market-20242030)
[The future of Jobs report 2025](https://www.weforum.org/publications/the-future-of-jobs-report-2025/)

## Setup

### Backend
1. Navegue até a pasta `backend`: `cd backend`
2. Crie um ambiente virtual: `python -m venv venv`
3. Ative o ambiente virtual: `source venv/bin/activate` (Linux/macOS) ou `venv\Scripts\activate` (Windows)
4. Instale as dependências: `pip install -r requirements.txt`
5. Rode o servidor: `python app.py`

### Frontend
1. Navegue até a pasta `front`: `cd front`
2. Instale as dependências: `npm install`
3. Rode o servidor de desenvolvimento: `npm run dev`
4. Abra o navegador em `http://localhost:3000/`
